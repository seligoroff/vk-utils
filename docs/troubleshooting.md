# Решение проблем

## Ошибка "Invalid token"
- Проверьте правильность токена в `.env`
- Убедитесь, что токен не истек
- Проверьте права доступа токена

## Ошибка "Access denied"
- Убедитесь, что токен имеет необходимые права (`wall`, `groups`, `audio`)
- Проверьте, что группа/пользователь доступны для токена

## Ошибка "Too many requests"
- Увеличьте задержку между запросами (`--delay`)
- Подождите некоторое время перед повторным запуском

## Файл `vk-groups.csv` не найден
- Создайте файл `resources/vk-groups.csv`
- Добавьте в него URL групп, по одной на строку
- Или используйте `make vk-groups-file` для автоматического создания

## Ограничения VK API

VK API имеет ограничения на количество запросов:
- **Rate limiting:** Инструмент автоматически добавляет задержки между запросами
- **Лимиты:** При превышении лимитов API вернет ошибку - нужно подождать

Рекомендуется:
- Использовать задержки между запросами (параметр `--delay`)
- Не запускать несколько команд одновременно
- Использовать кеш (`--cached`) для повторного просмотра результатов

## Поддержка

При возникновении проблем:
1. Проверьте настройки в `.env`
2. Убедитесь, что токен VK API валиден
3. Проверьте логи Laravel в `storage/logs/`
