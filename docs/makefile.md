# Makefile команды

Проект включает Makefile для упрощения работы с основными командами. Просмотреть все доступные команды:

```bash
make help
```

## Основные команды

```bash
# Полная настройка проекта (установка + ключ + миграции)
make setup

# Установка зависимостей
make install

# Обновление зависимостей
make update

# Запуск миграций
make migrate

# Очистка кешей
make clean

# Запуск тестов
make test
```

## Команды для работы с VK API

```bash
# Проверить последние посты в группах
make vk-check

# Получить информацию о группах
make vk-groups-info
```
